# Climate-Risks-Emissions
This project presents a cross-country analysis of climate risks, CO₂ emissions, and heat-related mortality.  It highlights regional disparities and links between environmental stress and public health outcomes.
## Data
- Cross-country climate and emissions indicators
- Health outcome data (heat-related mortality)
## Tools Used
- Power BI
## Key Insights
- Climate risks vary significantly across regions
- High-emission regions show greater environmental stress
- Vulnerable countries experience higher heat-related mortality
